# Nivo Money - Admin Application

Administrative operations dashboard for Nivo Money. Secure, desktop-optimized interface for compliance officers, support staff, and system administrators.

## Overview

The admin app is a separate, security-hardened application designed specifically for internal staff operations.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server (runs on port 3001)
npm run dev
```

## Security Features

- 2-hour session timeout
- Activity-based auto-logout
- All routes require authentication
- Audit logging enabled
